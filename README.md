# ts-typeorm-crud

## Important:

  - Initialize your web development environment such as WAMPSERVER or XAMPP.
  
 
Open a command line, locate the project address and run the following npm scripts, 

in developer mode or in production mode:

  
## Development mode:
```
  npm run dev
```

## Build:
```
  npm run build
```

## Production mode:
```
 npm run start
```
