# ts-typeorm-crud

## Important:

  - Initialize your local web server ( WAMP o XAMP)
  
  
Open a command line, locate the project address and run the following npm scripts, 

in developer mode or in production mode:

  
## Development mode:
```
  npm run dev
```

## Build:
```
  npm run build
```

## Production mode:
```
 npm run start
```
